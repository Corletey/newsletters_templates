How to use these. Each newsletter is a standalone HTML file using only inline styles — copy the file’s source into your mailing platform’s “paste HTML” field. Image placeholders are marked [ PHOTO ]; swap them for <img> tags pointing to your hosted images (mail clients need absolute URLs). The partner logos, the Mastercard Foundation line and all copy are editable directly in the file. Colours, kente strip and pattern are drawn from your reference artwork.
